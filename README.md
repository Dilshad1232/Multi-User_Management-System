# Multi-User Management System (MUMS)

A simple **web-based Multi-User Management System** built with **Laravel**.  
It allows multiple users to register, log in, and manage their profiles, while an admin can manage all users.

## Features

- Admin and User roles with separate dashboards
- User profile management with photo upload
- Admin can view, edit, and delete users
- Responsive design
